Copypaster

Скорпированный в буфер обмена текст, скрипт ставит в инпутные поля на веб странице. Проставить id инпутных полей формы можно в файле settings.js.
 

 
• Не работает в private mode Firefox

• about:debbuging   ->   This Firefox    ->    Load Temporary add-on

• Загрузка manifest.js из папки copypaster
